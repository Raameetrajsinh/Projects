About This Project

This web application is built using Flask, a simple and beginner-friendly Python web framework. The goal of this project is to understand the basics of web development using Flask.

In this Project:

  • Set up a Flask application.
  • Create routes.
  • Use HTML templates with Jinja2.
  • Handle user input through forms.
  • Work with a database using SQLAlchemy and SQLite.
